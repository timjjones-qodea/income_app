Guide

How to use Retirement Income

The app builds a household natural-income view from real AJ Bell holdings, actual cash receipts and optional AIC planning data.

Start importing

The job of the app

Use it to see what Wendy and Tim actually hold, what dividend and interest income those holdings produced, which wrapper the income landed in, and what looks repeatable for retirement planning.

The workflow

  1. 1
    Import AJ Bell portfolio

    Required first. This creates current holdings, quantities, tickers and market values.

  2. 2
    Import AJ Bell cash statement

    Required for actual income. Use roughly the previous 12 months for each account.

  3. 3
    Optionally import AIC

    Use the AIC Income Builder export as the repeatable investment-trust income baseline.

What each page is for

Dashboard

Household overview: portfolio value, actual trailing income, planning income and data quality warnings.

Imports

Stage, validate, commit and roll back CSV files. Start here for every data refresh.

Holdings

Current holdings and planning income. The Source column explains where each estimate came from.

Income

Actual dividend and interest receipts from AJ Bell cash statements, grouped by calendar and UK tax year.

Securities

Security master data, unmatched row mapping and manual income assumptions.

Reconciliation

Control page comparing actual receipts with imported dividend-event data where available.

Built-in AJ Bell cash-statement rules

BALANCE B/F *

Opening balance carried forward. Kept for audit context, not treated as income or movement.

Account charge … account code

Coded as account charge. The AJ Bell account code, for example ABWD2VD, is extracted for cash-activity reporting.

Cash Withdrawal

Coded as cash withdrawal so drawings can be reported separately from income.

Gross interest to date

Coded as gross interest and included in the Income page interest total.

Dividend quantity + name

Coded as dividend and matched to the imported security holding.

These rows no longer appear as unmatched securities unless they are genuine dividend, buy or sell rows requiring a security match.

Planning income priority

  1. Manual annual dividend per share.
  2. Manual forward yield.
  3. AIC Income Builder portfolio export.
  4. AJ Bell actual trailing-12-month receipts, including specials.
  5. Imported dividend events.
  6. Asset-type fallback yield.

If a holding is using fallback yield, treat the number as a placeholder and add better data when available.

Practical refresh cadence

  • Monthly or after trades: refresh AJ Bell portfolio CSVs.
  • Monthly or quarterly: refresh AJ Bell cash statements.
  • Monthly or quarterly: refresh AIC exports for covered investment-trust portfolios.
  • Annually: export income reports for tax-year review.
