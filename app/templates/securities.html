Securities

Manual assumptions override imported dividend data and fallback yields.

When to use this page

Use Securities when an income row did not match a holding, or when the automatic planning source is not the right basis for the future. Manual dividend-per-share and manual yield assumptions take top priority in Holdings.

  • Map unmatched AJ Bell descriptions to the correct security.
  • Add manual assumptions for securities outside AIC coverage.
  • Override stale or distorted income data.

Built-in AJ Bell cash-statement rules

BALANCE B/F *

Coded as opening balance. It is retained for audit context, but it is not treated as income, a charge or a security movement.

Account charge … ABWD2VD

Coded as account charge. The AJ Bell account code is extracted from the description so charges can be reported by account code later.

Cash Withdrawal

Coded as cash withdrawal. It is not a security-matching issue.

Gross interest to date

Coded as gross interest and included in Income as account interest.

Dividend quantity + name

Coded as dividend and matched to the imported holding by security name.
